TemporalOroboros updated:

  • A recent firmware update reenabled the screens on Nanotransen brand computers.
  • Bloodsplatter no longer experiences phantom limb syndrome on people missing both of their arms.
  • Recent budget cuts have resulted in accessories no longer having magnets strong enough to pull bloodstains directly out of clothing. Washing machine usage has jumped 50% over the last week.

Timberpoes updated:

  • Plexus Access Management app has had the Basic tab renamed to Trim to indicate that these accesses are available because of the card's trim. It also has an access counter so players can see they're able to change it.
  • Plexus Access Management app has had the logic for displaying the used/available accesses for any given wildcard inverted. It now shows Used/Max instead of Available/Max.
  • Wallets no longer show redundant examine text.

Time-Green updated:

  • Plumbing reaction chambers now have connects for acidic (yellow) and basic (green) buffers! You can set the UI to indicate at what pH the reaction chamber is supposed to insert a buffer into the reaction.
  • Adds plumbing bufferers. Place a bunch of them next to each other, set a threshold amount with right click and screwdriver them. Once all are above the threshold, only then will they start providing chems. Great for waiting steps and carefully dosing amounts!
  • The reaction chamber no longer pulls specific chems, and instead has a set volume it works towards before being able to react.
